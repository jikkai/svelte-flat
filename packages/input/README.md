# Input 输入框

文本输入框组件。

## Attributes
|参数|说明|类型|可选值|默认值|
|---|---|---|---|---|
|value|值|String|-|-|
|type|类型|String|text,password|text|
|status|状态|String|default,error,success|default|
|placeholder|占位文本|String|-|-|
|disabled|是否禁用|Boolean|-|false|

## Events
|事件名称|说明|回调参数|
|---|---|---|
|onfocus|获得焦点时触发|event|
