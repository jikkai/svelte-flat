## 属性

| 参数 | 说明 | 类型 | 可选值 | 默认值 |
| ------ | ------ | ------ | ------ | ------ |
| type | 设置输入框原生类型 | string | `text` `password` | `text` |
| size | 设置输入框尺寸 | string | `small` `large` `huge` | - |
| value | 设置输入框的值 | string | - | - |
| disabled | 设置输入框禁用状态 | boolean | - | false |
