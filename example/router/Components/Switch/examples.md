## 属性

| 参数 | 说明 | 类型 | 可选值 | 默认值 |
| ------ | ------ | ------ | ------ | ------ |
| type | 设置按钮类型 | string | `warning` `danger` `success` `info` | - |
| size | 设置按钮尺寸 | string | `small` `large` `huge` | - |
| shape | 设置按钮边框形状 | string | `square` | - |
| checked | 设置按钮选中状态 | boolean | - | false |
| disabled | 设置按钮禁用状态 | boolean | - | false |

## 事件

| 事件名 | 说明 | 回调参数 |
| ------ | ------ | ------ |
| click | 触发点击事件 | Event |
